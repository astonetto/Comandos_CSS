# Comandos CSS

## Altura

```js
height: 100%;
```

## Borda Lateral

```js
border-right: 1px solid #ddd;
```

## Cor de Fundo

```js
background-color: #ededed;
```

## Itens um abaixo do outro

```js
flex-direction: column;
```

## Largura

```js
width: 100%;
```

## Maxima Largura

```js
max-width: 415px;
```

## Para que um item, fique do lado do outro

```js
display: flex;
```

## Dos itens que tem dentro do header, vai ter espaços entre eles, ou seja joga um para um lado o outro para o outro e tera um espaço entre eles

```js
justify-content: space-between;
```

## Centraliza Verticalmente, os botões e o icone

```js
align-items: center;
```

## Distancia entre os lados, 0, encima e embaixo, 15px distancia entre os dois lados

```js
padding: 0 15px;
```

## Fique redonda os lados

```js
border-radius: 20px
```

## Pode clicar neste item

```js
cursor: pointer;
```

## Borda botton

```js
border-bottom: 1px solid #eee;
```

## Pegar o espaço disponivel

```js
flex: 1;
```

## Barra de rolagem, quando tiver bastante itens

```js
overflow-y: auto;
```
